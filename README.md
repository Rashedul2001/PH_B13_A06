# 🌟 DigiTools-Platform
This is an Assignment project from Programming Hero where I had to make a beautiful webpage using React and implement buy, remove and  calculation functionality. But above all it is for learning the React Library.

## 🛠️ Technologies Used
- React
- Tailwind CSS 
- avaScript (ES6+)
- React-Toastify
- Daisy UI

## ✨ Features
1. **Responsive Design** - Works seamlessly across desktop, table and mobile. 
2. **Scalable Architecture** -  Built with reusable components and clean code practices.
3. **Functionality** - Provide attracting Toast message, shows Proper Buying and deleting functionality and show correct Total price and Product quantity.
